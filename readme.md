

# Django AI Chat App

A simple AI-powered chat application built using Django, JavaScript, HTML, CSS, and the Gemini API.

## Features

- AI chat interface
- Dark theme UI
- Responsive design
- Django backend
- JavaScript frontend integration
- Gemini API integration
- Error handling

## Technologies Used

- Python
- Django
- JavaScript
- HTML
- CSS
- Gemini API

## Installation

1. Clone the repository
2. Install dependencies
3. Run migrations
4. Start the Django server

```bash
python manage.py migrate
python manage.py runserver
```
